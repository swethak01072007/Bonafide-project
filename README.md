# Bonafide-project
The Bonafide Automation System is a web-based application developed to digitize and automate the bonafide certificate request process within the institution. It replaces the traditional manual paperwork with a secure, role-based workflow, allowing students to submit requests online 
By automating the bonafide certificate workflow, the system reduces processing time, minimizes manual errors, improves transparency, and provides a scalable solution that can be implemented across the entire institution.
